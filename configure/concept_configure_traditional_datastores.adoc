= 配置传统数据存储库和虚拟机
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用ONTAP工具在vCenter Server中配置数据存储库和虚拟机。ONTAP工具配置的数据存储库和虚拟机显示在ONTAP工具的信息板上。这样，您就可以轻松地监控和管理这些数据存储库和虚拟机。
